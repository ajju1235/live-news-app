# 📰 Live News Web App

A responsive **Live News Web Application** that fetches real-time news articles using a public news API.
Users can search for news topics and browse categories like IPL, Finance, and Politics in a clean and responsive interface.

🔗 **Live Demo:**
https://ajju1235.github.io/live-news-app/

---

## 🚀 Features

* 🔍 **Search News** by keywords
* 📰 **Real-time news articles** using API
* 📂 **Category navigation** (IPL, Finance, Politics)
* 📱 **Responsive UI** for different screen sizes
* 🖼️ News cards with:

  * Title
  * Image
  * Description
  * Source
  * Published date
* 🔗 Click on a news card to open the full article

## ⚙️ How to Run the Project Locally

1. Clone the repository

```
git clone https://github.com/ajju1235/live-news-app.git
```

2. Open the project folder

```
cd live-news-app
```

3. Open **index.html** in your browser

Or run using **VS Code Live Server**.
