# Live News Web Application

A simple real-time news web application built using HTML, CSS, and JavaScript.

## Features
- Fetches latest news using News API
- Category-based news filtering
- Dynamic news cards using JavaScript
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- REST API (News API)
